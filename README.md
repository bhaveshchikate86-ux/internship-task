AI Website Copy Project – Cafe Aroma
Business Chosen
The business selected for this project is Cafe Aroma, a modern and cozy cafe located in Pune, Maharashtra.

Cafe Aroma is a popular spot for students, friends, and families to relax and enjoy quality time in a comfortable environment.

The cafe offers a variety of food items including coffee, sandwiches, burgers, desserts, and refreshing beverages. It provides both dine-in and takeaway services.

Prompt Logic
The prompt was carefully designed to generate a professional and complete website copy.

It included:

- Cafe name and location (Pune)
- Homepage headline and subheadline
- About section content
- Menu and services details
- Call-To-Action (CTA) section
- Customer-focused tone and simple language

The goal was to generate realistic, business-focused, and customer-friendly website content instead of generic AI text

Tool Used
This project was created using ChatGPT.

The structured prompt was used to generate website content.
Additional refinements were made to improve clarity and quality, making the output suitable for real business use.

Project Summary
This project demonstrates prompt engineering skills by creating AI-generated website content for a real cafe business.

The generated content includes homepage copy, services page content, and CTA sections, making it ready for presentation to a business owner or client.
